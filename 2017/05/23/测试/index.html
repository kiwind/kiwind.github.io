尊敬的水城三街区全体业主：
您们好，今日三街区业委会临时筹备组已经向沿江街道正式递交成立业委会相关材料，街道已受理申请，另，社区已派工作人员积极介入小区成立业主委员会相关工作，经确认，物业及开发商也正在准备成立业主委员会需要的材料，业委会成立已正式拉开序幕！因前期业委会临时筹备组开展工作需配备必要的办公用品及费用开支，如：录音笔、摄录机、锦旗等，现向全体业主募集资金（以户为单位）：30-50元/户，筹备资金广大业主可通过各楼长进行缴费。希望全体业主积极参与进来，为业委会成立作出各自的贡献。
在此，业委会临时筹备组承诺：所购资产所有权归属全体业主，所有资金收支公开透明，接受业主监督。
